Celebrity DOB Service

The Ai Core Celebrity DOB service.

Endpoints

  • DOB of a celebrity
    GET /api/dob/{first_name}?last_name={last_name}&city={city}

    Parameters
    • Required: first_name={first_name} - The first name of the celebrity you want to search
    • Required: last_name={last_name} - The last name of the celebrity you want to search
    • Optional: city={city} - Boolean. If True, the JSON file will return the birth place of the celebrity
